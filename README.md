# EV Market Intelligence Dashboard

## Overview
This Power BI dashboard analyzes electric vehicle market trends, vehicle performance, battery efficiency, customer ratings, and annual sales insights.

## Tools Used
- Power BI
- DAX
- CSV Dataset
- Data Visualization

## Dashboard Features
- Sales Analysis
- Battery vs Range Analysis
- Customer Ratings
- Vehicle Distribution
- Country-wise Sales
- Interactive Slicers

## Files Included
- EV Dashboard.pbix
- EV Dataset.csv
- Dashboard Screenshot
